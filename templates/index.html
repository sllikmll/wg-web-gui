WireGuard Web GUI

Управление серверами, клиентами, диагностикой, backup и импортом WireGuard / wg-easy

Серверы

Контейнер wg-easy, interface, config path, UDP port, subnet и DNS определяются автоматически из remote config. Удаление сервера удаляет его только из базы панели.

Клиенты

При добавлении делается backup remote config, запись peer в config и restart только WireGuard-контейнера.

Мониторинг и трафик

Online, last handshake, endpoint, RX/TX и накопительный трафик по клиентам.

Журнал событий

Sync, polling, backup restore, import и ошибки.
